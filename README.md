## Installation 

1. Add the latest version of package to your pubspec.yaml (and run`dart pub get`):
```yaml
dependencies:
  flbootstrap: ^0.0.1
```
2. Import the package and use it in your Flutter App.
```dart
import 'package:flbootstrap/flbootstrap.dart';
```
